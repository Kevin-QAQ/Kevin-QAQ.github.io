解决无法打开 TeX Live 的问题

问题

成功安装 TeX Live 后，仅在首次使用时可以正确打开 TeX Live，之后想再次启动程序无反应，或有命令行窗口闪现后秒退，重启电脑不解决问题。

暂时的解决方案

  1. 首先找到 TeX Live 的安装位置，我是默认安装位置，在 C:\texlive\2020\bin\win32 路径下。

  2. 找到 tlaunch.exe 程序，可以将其固定到开始屏幕，方便以后使用。

  3. 双击打开该程序，菜单栏选择 Tools，选择 Re-initialize TeX Live，点确认、确认。

  4. 最后点击第一个图标 LaTeX Editor 就可以愉快的使用 TeX Live 了。

之后想使用 TeX Live 时，我们只用在开始屏幕中打开 tlaunch，执行 3、4 步即可。
